Google Talk

Description

Homepage: http://www.google.com/talk/
Type: XMPP-based, proprietary
Identity: Jabber ID (JID), e.g. usr@a.b.c/home with "home" being a "resource", which distinguishes the same user when logged in from multiple locations (simultaneously)
Interoperability: XMPP, LiveJournal, Gizmo

Apart from using and being compatible with XMPP, Google Talk also has proprietary elements.

Note that "a Gmail account is required to use Google Talk or other download clients" (says Google).
