Menu bar icons

.AdiumMenuBarIcons Xtras provide alternate menu bar icons for Adium.

Bundled dock icons

By default, Adium contains two menu bar icons that can be configured from Adium's Appearance preferences pane:

  • Adiumy
  • Adiumy Flashy

Installing dock icons

More menu bar icons are available at the AdiumXtras website, and can be installed by clicking the install link on the Xtra's page. To install an .AdiumMenuBarIcons file that you have already downloaded, simply doubleclick it. In either case, Adium will launch and put it into the correct place for you.

Creating a dock icon

Information on making custom menu bar icons is available at Creating menu bar icons.
