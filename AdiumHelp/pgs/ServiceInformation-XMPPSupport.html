XMPP (Jabber) in Adium

This page deals with XMPP (historically known as "Jabber") specialities.

Commands in multi-user chats (MUCs)

Adium supports the following commands in multi-user chats.

Command Meaning
/affiliate owner <nickname> sets user affiliation to "owner" / gives user full permission over a room
/affiliate admin <nickname> sets user affiliation to "admin"
/affiliate member <nickname> sets user affiliation to "member"
/affiliate outcast <nickname> sets user affiliation to "outcast" / bans user
/affiliate none <nickname> sets user affiliation to "none"
/role moderator <nickname> sets user role to "moderator"
/role participant <nickname> sets user role to "participant"
/role visitor <nickname> sets user role to "visitor"
/role none <nickname> sets user role to "none" / kicks user
