List themes

A .ListTheme AdiumXtra defines the colors and other visual information which appear in your Contact List, including but not limited to:

  • Highlight color
  • Alternating grid
  • Window background (Normal and Borderless contact list window styles only)
  • Status-based coloration
  • Status text color (if extended status is displayed), such as colors displayed for Online, Away, or Idle contacts
  • Wvent text color, such as colors displayed for recently signed-on contacts or contacts who are currently typing to you
  • Group background colors

Installing list themes

More list themes are available at the AdiumXtras website, and can be installed by clicking the install link on the Xtra's page. To install an .ListLayout file that you have already downloaded, simply doubleclick it. In either case, Adium will launch and put it into the correct place for you.
