Dock icons

.AdiumIcon Xtras provide alternate dock icons for Adium.

Bundled dock icons

By default, Adium contains six dock icons that can be configured from Adium's Appearance preferences pane:

  • Adiumy Black
  • Adiumy Blue
  • Adiumy Green
  • Adiumy Purple
  • Adiumy Red
  • Adiumy Yellow

Installing dock icons

More dock icons are available at the AdiumXtras website, and can be installed by clicking the install link on the Xtra's page. To install an .AdiumIcon file that you have already downloaded, simply doubleclick it. In either case, Adium will launch and put it into the correct place for you.

Creating a dock icon

Information on making custom dock icons is available at Creating dock icons.
