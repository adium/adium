Mentions

Adium provides indication of when your user name (and some other text you possibly specified - see below) has been mentioned in a group chat. Primarily it does so by displaying a grey line in the scrollbar (not customizable) but Adium also provides a way for message styles to apply custom styling to messages that contain a mention.

Specifying mentions

Mentions can be specified in Preferences → Advanced → Mention. By default, Adium highlights messages that include your username but you can add other notification strings to be highlighted.

Scrollbar indicators

  • When your user name (or what you decide to highlight) is mentioned in a chat, a dark grey horizontal line is added.
  • You can access the "Jump to Previous Mark" or "Jump to Next Mark" from the Display menu or by right-clicking the scroll bar.

Mention count

Mentions are also indicated by the color of the tab's unread mention count changing to red.

  • Mention count preferences can be changed in Preferences → Advanced → Message Alerts.

In-chat indicators

Adium allows message styles to provide their own, in-chat indicators for mention. All message styles bundled with 1.4 (or newer) provide a different, style-integrated indication for mentions that indicate the specific message that contains a message.

  • Gone Dark uses a colored background (matches variant color).
  • minimal_mod uses a dark block at the left edge (matches header color).
  • Mockie uses a "•" in the margin (may be left or right) (matches variant color).
  • Renkoo uses a colored background (matches variant color).
  • Smooth Operator uses a star in the right margin (matches text color).
  • Stockholm uses a mention icon to the left of the relevant message (mention icon will replace a buddy icon if the user has chosen to display buddy icons) (matches variant color).
  • yMous uses a colored block at the right edge of the relevant message (between the message and the timestamp if the timestamp is visible) (matches variant color).
