Adium and Mac OS X sleep mode

How Mac OS X sleep mode affects Adium

Mac OS X automatically disconnects all network connections when you put your Macintosh to sleep. As a result, Adium can't stay connected to your accounts. Adium will automatically attempt to reconnect your accounts when the machine is awakened.

If Adium is not automatically reconnecting after the machine is awakened, check these items:

  • Do you have a network connection? You may have lost any AirPort connections if you are using a PowerBook which has moved since going to sleep.
  • Are you using the newest version of Adium? Recent versions have significantly improved reconnection upon awakening.
