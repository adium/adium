Gizmo5

Description

Homepage: http://gizmo5.com/
Type: XMPP-based, proprietary
Identity: Jabber ID (JID), e.g. username@chat.gizmoproject.com
Interoperability: XMPP, LiveJournal, Google Talk

Apart from using and being compatible with XMPP, Gizmo5 also has proprietary elements.

Gizmo's chat system is simply XMPP, so all you need to do is add the account as a Jabber account. For example, if your Gizmo Project username is "jdoe", your Jabber ID is "jdoe@chat.gizmoproject.com".
