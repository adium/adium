List layouts

A .ListLayout AdiumXtra defines the appearance of your Contact List, including but not limited to:

  • Font
  • Contact icon presence, size, and location
  • Service icon presence and size
  • Status icon presence and size
  • Spacing

Bundled list layouts

Installing list layouts

More list layouts are available at the AdiumXtras website, and can be installed by clicking the install link on the Xtra's page. To install an .ListLayout file that you have already downloaded, simply doubleclick it. In either case, Adium will launch and put it into the correct place for you.
