Sound sets

.AdiumSoundSet Xtras define a set of sounds for use when particular events occur. A soundset as a whole can be chosen, replacing all existing configured sounds with those in the set, or the sounds from within multiple sets can be chosen individually for events of the user's choice.

Bundled sound sets

By default, Adium contains five soundsets that can be configured from Adium's Events preferences pane:

  • Adium
  • AIM
  • iChat
  • None
  • TokyoTrainStation

Installing sound sets

More sound sets are available at the AdiumXtras website, and can be installed by clicking the install link on the Xtra's page. To install an .AdiumSoundSet file that you have already downloaded, simply doubleclick it. In either case, Adium will launch and put it into the correct place for you.

Creating a sound set

Information on making custom sound sets is available at Creating sound sets.
