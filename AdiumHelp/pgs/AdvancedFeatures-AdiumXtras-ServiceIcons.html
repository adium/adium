Service icons

.AdiumServiceIcons Xtras provide alternate service icons for Adium.

Bundled service icons

By default, Adium contains only a single set of service icons that can be configured from Adium's Appearance preferences pane:

  • Aqua

Installing service icons

More emoticon sets are available at the AdiumXtras website, and can be installed by clicking the install link on the Xtra's page. To install an .AdiumServiceIcons file that you have already downloaded, simply doubleclick it. In either case, Adium will launch and put the Xtra into the correct place for you.
