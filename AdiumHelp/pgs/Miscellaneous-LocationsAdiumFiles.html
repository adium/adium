Locations of Adium files

This page lists the locations of all Adium-related files.

Note: The "Library" folder mentioned below is hidden in OS X versions 10.7 and newer. See this article for details.

Location of the application

Normally, your Adium should be placed in the following location:

  • Home Folder → Applications → Adium.app

Location of preferences

Adium places the majority of its preferences here:

  • Home Folder → Library → Application Support → Adium 2.0

There is also a preference file at:

  • Home Folder → Library → Preferences → com.adiumX.adiumX.plist

Location of saved transcripts

Message transcripts can be found in this directory:

  • Home Folder → Library → Application Support → Adium 2.0 → Users → Default → Logs

Location of user icons

Your own icons

Recent icons set via the icon picker (both for contacts and for yourself) are cached at ~/Library/Images/iChat Recent Pictures, and such recent icons are shared with iChat.

Icons of your contacts

Icons, once retrieved, are cached at ~/Library/Caches/Adium/Default. (The "Library" folder is hidden in OS X versions 10.7 and newer. See this article for details.)

Location of non-standard, installed Xtras

The Adium Xtras you have installed are stored in subfolder of the "Adium 2.0" folder mentioned above:

  • Home Folder → Library → Application Support → Adium 2.0
    • → Contact List - Contact list layouts and themes (.ListLayout, .ListTheme)
    • → Emoticons - Emoticon packs (.AdiumEmoticonSet)
    • → Menu Bar Icons - Menu bar icon packs (.AdiumMenuBarIcons)
    • → Message Styles - Message styles (.AdiumMessageStyle)
    • → PlugIns - Plugins (.AdiumPlugin)
    • → Scripts - Scripts (.AdiumScript)
    • → Service Icons - Service icon packs (.AdiumServiceIcons)
    • → Sounds - Sound packs (.AdiumSoundSet)
    • → Status Icons - Status icon packs (.AdiumStatusIcons)
