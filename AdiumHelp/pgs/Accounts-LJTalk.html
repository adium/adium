LJ Talk

Description

Homepage: http://www.livejournal.com/chat/
Type: Open standard (XMPP)
Identity: Jabber ID (JID), e.g. LJ-usr@livejournal.com/home with "home" being a "resource", which distinguishes the same user when logged in from multiple locations (simultaneously)
Interoperability: Google Talk, LiveJournal

LiveJournal is Six Apart's blogging service. Its XMPP-based instant-messaging service is called "LJ Talk".
